更新流程

约 132 字小于 1 分钟

2024-11-02

  1. 编辑内容

    • 使用 Visual Studio Code 编辑
    code ~/03_VuePress/vuepress/docs/
    • 预览网站效果
    pnpm run docs:dev

  2. 本地构建

    • 生成静态 HTML 文件
    pnpm run docs:build
    • 转移静态文件到 git 文件夹
    cp -r .vuepress/dist/* ~/07_Cloudflare_Pages/my_vuepress

  3. 上传

    • 进入 git 文件夹
    cd ~/07_Cloudflare_Pages/my_vuepress
    • 添加到暂存区
    git add .
    • 提交到本地仓库
    git commit -m "提交的更新信息"
    • 推送到 Github
    git push

  4. 部署

    • Cloudflare Pages 自动部署
